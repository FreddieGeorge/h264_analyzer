## Highlights

- Keeps `ZStreamEye.exe` in the portable/install root while grouping runtime
  files under `runtime/`.
- Fixed garbled buffering overlay text when browsing old frames.

## Changed

- Windows packages now use a root-level `ZStreamEye.exe` launcher.
- The Qt application and runtime DLLs/plugins are grouped under `runtime/`.
- Installer upgrades remove old root DLLs, old Qt plugin folders, and the
  previous `bin/` layout before installing the new root launcher/runtime layout.

## Fixed

- The old-frame buffering overlay now uses an ASCII status message to avoid
  garbled text on Windows/OpenGL text paths.

## Notes

- Installation remains interactive, not silent.
- The Windows installer is not code signed yet, so Windows may show "Unknown publisher".
