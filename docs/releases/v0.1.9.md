## Added

- GPL v3 license file (`LICENSE`) for proper open-source distribution.
- Direct H264 hex navigation from property tree summary fields.
- H264 summary fields now drive hex selection in the bitstream view.
- Access unit filters in the analysis list (All, Video only, Audio only, Diagnostics only).
- Packet hex evidence display for parsed access units.
- Documentation for H264 hex navigation feature.

## Changed

- Project version is now `0.1.9`.
- Property tree H264 summary rows now directly link to hex view selection.
- Bitstream hex dock now highlights byte ranges based on selected syntax fields.

## Notes

- The license file follows GNU GPL v3 terms.
- Hex navigation currently supports H264 SPS/PPS/slice header fields and selected macroblock syntax fields.
- Access unit filters allow users to filter the analysis list by media type or diagnostic status.
