## Highlights

- Fixed playback controls after a video reaches the end.

## Fixed

- After natural playback completion, Previous Frame and Next Frame remain
  available for cached/seeking navigation.
- The Play button now acts as Replay when decoding has finished and an open
  stream is still loaded.
- Stop is only enabled while a decode/playback worker is running.
- Windows installer upgrades now remove DLLs and Qt plugin folders left in the
  install root by older releases before installing the new `bin/` layout.

## Notes

- Installation remains interactive, not silent.
- The Windows installer is not code signed yet, so Windows may show "Unknown publisher".
