## Highlights

- Renamed the application and published artifacts to ZStreamEye.
- Added the ZStreamEye application icon for the app window and Windows executable.
- Added GitHub Releases based update checking.
- Published Windows portable ZIP and installer artifacts.

## Changed

- Windows packages now use the `ZStreamEye-*` naming scheme.
- The application version is `0.1.1`.
- Update checks now use the `FreddieGeorge/ZStreamEye` GitHub Releases feed.

## Notes

- The Windows installer is not code signed yet, so Windows may show "Unknown publisher".
- The current updater opens the release page or installer download in the browser; it does not silently install updates.
