## Highlights

- Added an About ZStreamEye dialog.
- Improved update failure and cancellation handling.

## Added

- `Help -> About ZStreamEye` now shows the app version, GitHub link, copyright,
  and open-source project note.

## Changed

- If an update installer exists but its SHA256 checksum asset is missing, the
  update dialog now explains that clearly and directs users to the release page.
- Download failures now suggest trying again and include the network error.
- Canceling an update download is treated as a normal user action instead of a
  warning.
- SHA256 mismatch handling explicitly removes the downloaded file and says so.

## Notes

- Installation remains interactive, not silent.
- The Windows installer is not code signed yet, so Windows may show "Unknown publisher".
