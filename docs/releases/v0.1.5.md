## Highlights

- Expanded the Open File dialog to common video containers and bitstream file extensions.
- Tidied Windows portable and installer layouts by moving runtime files under `bin/`.
- Improved the About dialog close behavior.

## Changed

- `File -> Open` now includes common video/container extensions such as MP4,
  MOV, MKV, AVI, TS, WebM, FLV, WMV, MPEG, 3GP, MP5, HEVC/H.265, and H.264 raw
  bitstreams.
- Portable packages now keep the root folder clean:
  - `README-RUN.txt`
  - `bin/ZStreamEye.exe`
  - runtime DLLs and Qt plugins under `bin/`
- Installer shortcuts now launch `bin/ZStreamEye.exe`.
- `About ZStreamEye` now has a proper Close button and can be dismissed through
  the window close control.

## Notes

- Installation remains interactive, not silent.
- The Windows installer is not code signed yet, so Windows may show "Unknown publisher".
