## Highlights

- Added direct update download actions to the update checker.
- Standardized release notes through `docs/releases/<tag>.md`.

## Added

- `Help -> Check for Updates` now offers `Download Installer` when a matching
  Windows installer asset is available on the latest GitHub Release.
- The update dialog can also open the portable ZIP asset when available.
- Release workflow now requires `docs/releases/<tag>.md` and publishes it as
  the GitHub Release body.
- Added release notes guidance in `docs/releases/README.md`.

## Changed

- Screenshot exports now default to `zstreameye-screenshot.png`.

## Notes

- Downloading the installer opens the browser/default download handler. The app
  still does not silently install updates.
- The Windows installer is not code signed yet, so Windows may show "Unknown publisher".
